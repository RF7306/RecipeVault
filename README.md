# RecipeVault

A Progressive Web Application built with **Ionic 7** and **Angular 17** (standalone components).

## Quick Start

```bash
npm install
ionic serve
```

## Stack

- Ionic 7 + Angular 17 (standalone)
- Capacitor 5 (Camera plugin)
- @ionic/storage-angular (IonicStorage)
- RxJS Observables + HttpClient
- SCSS with dark mode support

## Pages

| Page | Route |
|------|-------|
| Recipe Feed | `/home` |
| Recipe Detail | `/recipe/:id` |
| Meal Planner | `/meal-planner` |
| Shopping List | `/shopping-list` |
| Saved Recipes | `/my-recipes` |
